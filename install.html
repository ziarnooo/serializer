How to install

  1. 1
    Extract the ZIP

    Find serializer.zip in your downloads and extract it to any folder.

  2. 2
    Open Chrome extensions

    In Chrome, navigate to chrome://extensions

  3. 3
    Enable Developer mode

    Toggle the "Developer mode" switch in the top-right corner of that page.

  4. 4
    Load unpacked

    Click "Load unpacked" and select the folder you just extracted.

  5. 5
    Pin the extension

    Click the puzzle icon in Chrome's toolbar and pin Serializer for quick access.
